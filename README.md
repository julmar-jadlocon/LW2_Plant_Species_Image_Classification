**Brief Description of the Project**

This project focuses on developing an image classification model using Teachable Machine to identify different non-native cultivated plant species. The system is trained using a dataset of plant images and is capable of predicting the class of a plant based on its visual features such as leaves, flowers, and overall structure.

**Purpose of the Image Classification Model**

The purpose of this project is to:

Automatically identify and classify 20 non-native cultivated plants
Assist students and users in recognizing plant species through images
Demonstrate the application of machine learning in plant classification
Provide a simple and accessible tool for educational and research purposes
